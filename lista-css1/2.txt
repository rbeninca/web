Seletores css são a forma de vincular uma regra css a elementos HTML, eles funcionam por especifidade de forma que  a regra css com seletor mais especifico para um determinado elemento é que tem a regra aplicada a ele. 
Os tipos de seletores CSS são : 

* : universal 
elemento: permite definir ou selecionar os elementos ao qual a regra será aplicado de acordo com nome do elemento. 
Exemplo: 
  h1{ border:1px solid black}
 
id : Seletor por identificador unico, permite identificar ou especificar unicamente um elemento HTML por meio da propriedade id definida para o elemento HTML. Exemplo : 

<style> 
   #p01{color:red}
</style>
<body>
    <p id="p01"> Lorem </p>
</body>
